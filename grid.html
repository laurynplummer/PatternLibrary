Responsive Grid System

For this pattern library a Flexbox base grid system will be presented. Flexbox-based grids will help the repsonsivitiy and structure of your website.

Introduction

The flex-box grid is based on 12 columns.For each row you can have up to 12 columns.

Grid Structure

Container

Provides stucture to the website. Placed in the <body> tag of the document.

Section

while adding padding between other content of the page

        <body>
    <div class="container">
    
    </div>

Row

Used to contain all of the columns. 

         <div class="row">
        <p>** Put Columns Here**</p> 
 </div>

Columns

The column class is contained inside a row. The column class provides padding brtween eahch column on the grid.

             <div class="row">
    <div class="column">
        <p>** Put Your Content Here**</p> 
    </div>
  </div>

Multiple Columns

You can have more than on column contained in a row, as long as it is divisible by 12

These are the combinations that will work in a grid

  • One Column
  • Two Columns
  • Three Columns
  • Four Columns
  • Six Columns
  • Tweleve Columns
