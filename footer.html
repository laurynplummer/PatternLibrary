Footer

The footer is another way to navigate around the website.

Usage

Using the <footer> tag,you can provide an alternate way of naviagtion. Using the <li> tags, you can link to more pages around your site.

Example Links
  • Home
  • About
Social Media Links
Footer

In order to organise structure this footer, you can use columns and rows to allign the content.

            
                <footer>
                <div class="row">
                    <div class="footer-column">
                        <h5>Example Links</h5>
                            <ul>
                                <li><a href="#!">Home</a></li>
                                <li><a href="#!">About</a></li>
                            </ul> 
                        </div>
                    <divclass="footer-column">
                        <h5>Social Media Icons</h5>
                            <a href="#!"><i class="fab fa-twitter"></i></a>
                            <a href="#!"><i class="fab fa-facebook-f"></i></a>
                            <a href="#!"><i class="fab fa-instagram"></i></a>
                    </div>
                    <div class="footer-column">
                        <h5>Footer</h5>
                          <p>In order to organise and structure this footer you can use columns and rows to 
                        allign the content.</p>
                     </div>
                </div>
            </footer>
