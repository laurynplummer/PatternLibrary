Buttons

Buttons should be used throughout a site where it is appropriate. Some buttons will have different uses, which will be explained throughout the documentation.

Button Components

Usage

Default Button

This is a default button, which can be used for anything. There might be different uses for this button but the general style and interaction will more than likely stay the same.

                <button"btn">Button</button>

Button Usage

This type of button may change for a usage. For example, Anchor, Button, Reset or Input.

                
                <button"btn">Button</button> 
<input type="submit" class="btn" value="Submit">
<a class="btn">Anchor Button</a>

Styles

Icon Button

Some buttons may require an icon, which can be added to the default and usage buttons.

Icon Button

Filled Buttons

Filled buttons can be used throughout the website. The colour can be changed in the CSS

            <button class="btn-fill1">Button</button>

Outlined Buttons

Outlined buttons provide an minimalistic effect, but can still add colour to your website.

            <button class="btn-out1">Button</button>

Rounded Filled Buttons

You may require buttons that are shaped differently 

                        <button class="btn-shape1">Button</button>

Width Button Filled

You may require buttons that span the width of the page, i.e menu buttons

                        <button class="btn-widthfill1">Button</button>

Width Button Outlined

Or you may require full width buttons that are just outlined

                <button class="btn-widthout1">Button</button>
